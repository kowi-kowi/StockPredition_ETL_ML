# StockPredition_ETL_ML

**Project Goal**
The goal of this project is to build an end-to-end data pipeline that collects market-related text data, processes it automatically (ETL), performs sentiment analysis, and uses the extracted sentiment features to predict short-term stock market movements using machine learning models.
This project demonstrates skills in:
- Data Engineering (ETL, automation)
- Natural Language Processing (NLP)
- Machine Learning
- Time-series & financial data analysis
- MLOps fundamentals
